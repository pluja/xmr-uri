Monero URI Generator

API Usage

Add these parameters to the URL:

  • ?rawqr - Returns QR code as PNG image
  • ?raw - Returns raw Monero URI as text
  • Parameters: address, tx_amount, tx_payment_id, recipient_name, tx_description

Example: index.html?address=44AFFq5kSiGBoZ...&tx_amount=1.5&rawqr
